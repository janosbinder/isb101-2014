UNIX Tutorial

Starting

Download the course material from the following link:

  • https://www.dropbox.com/s/6fwlbw5g1pctw9u/unixtut.tar.gz?dl=0

Copy the file to your home directory: % cp unixtut.tar.gz ~

change to your home directory: % cp ~

Unpack the tarball: % tar -zxvf unixtut.tar.gz

Now you should see a directory unixtut in your home directory.

Open the file index.html in your home directory in a web browser, e.g. firefox
